DOSBoxGTK

A GTK frontend for DOSBox developed in C++

Download .zip Download .tar.gz View on GitHub

Introduction

DOSBoxGTK is a DOSBox frontend for Gnu/Linux systems and the Gnome desktop developed in C++ using the GTKmm, LibXML++ and cURLpp libraries.

The user interface is very similar to the one used by DBGL.

Also provides a button to retrieve game information from MobyGames website when creating or editing a game profile.

Building dependencies

in order to build DOSBoxGTK you will need:

  • GNU C++ compiler (>= 4.8)
  • CMake (>= 2.8.12)
  • gtkmm (>= 3.10.1)
  • libXML++ (>= 2.36.0)
  • cURLpp (>= 0.7.3)

Building instructions

This instructions has been tested in an Ubuntu Trusty 64 bits Gnu/Linux distribution.

  1. Download the source from this page.

  2. Uncompress the downloaded file and go to the directory where the uncompressed files are.

  3. Create a folder for the building with a name, for example, like build and enter it:

    $ mkdir build
$ cd build

  4. Use CMake to configure the building environment:

    $ cmake -DCMAKE_BUILD_TYPE=RELEASE ..

    The code above will create the release version of DOSBoxGTK, which can be installed. If you want to build the debug version just use this:

    $ cmake ..

  5. If you configured the building environment to create the release version of DOSBoxGTK you need to install it in the system:

    $ sudo make install

  6. If you want to uninstall DOSBoxGTK just go to the directory where you built DOSBoxGTK and use:

    $ sudo make uninstall
